# 🎨 Дизайн-концепция: Smart Heater Control Panel (v1.1)

## 1. Визуальный стиль (UI)
*   **Стиль:** Modern Dark Mode / Glassmorphism (эффект матового стекла).
*   **Цвета:**
    *   `#121212` — Основной фон (Глубокий черный).
    *   `#1E1E1E` — Карточки и виджеты (Темно-серый).
    *   `#FF5722` — Акцент (Нагрев / Оранжевый).
    *   `#2196F3` — Акцент (Ожидание / Синий).

## 2. Динамическая индикация состояний (Visual Feedback)
Интерфейс меняет поведение в зависимости от данных из `SystemState`:

*   🔥 **РЕЖИМ НАГРЕВА (Relay == true):**
    *   Кольцо: **Оранжевое**.
    *   Эффект: "Breathing Glow" (плавная пульсация яркости тени вокруг кольца).
*   💤 **РЕЖИМ ОЖИДАНИЯ (Relay == false):**
    *   Кольцо: **Синее / Бирюзовое**.
    *   Эффект: Статичное свечение (покой).
*   🚨 **АВАРИЙНЫЙ РЕЖИМ (Critical Event):**
    *   Кольцо: **Красное**.
    *   Эффект: "Emergency Strobe" (резкая, быстрая пульсация всего виджета).
    *   Триггер: `alive == false` (таймаут 65с) ИЛИ `sensor_ok == false`.

## 3. Техническая реализация
*   **Frontend:** HTML5, CSS3, Vanilla JS.
*   **Backend:** LittleFS для хранения ресурсов.
*   **Связь:** Fetch API для получения `/api/status` и отправки `/api/set_temp`.
*   **Диапазон:** Регулировка температуры расширена до **35.0°C**.
*   **Индикация:** Расчет статуса `alive` выполняется Бриджем для стабильности UI.
